Trading on News
The key issue is to establishing relation between stock news and future stock price. namely, we are try to quantify how news impacts stock price. The advantage of our stragety is that users do not need to label press, as the system can automatically discriminate it. And this stragety is something like an event-driven strategy and sentiment analysis mixture. In addition Beside using the vector space model to make representation of the document and news, we can also use a named entity (such as organization name), phrases (such as event name like Merge&Acquisitions), etc. If we use event name like Merge&Acquisitions which may appear in news, then our strategy can be understood as that we attempt to find how event ike Merge&Acquisitions affect stock price.
